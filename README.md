# To-Do List Web Application

This project is a robust To-Do List web application built using JSP, Servlets, Java, SQL, and Materialize CSS. 
It provides a seamless user experience with modern design elements and a range of features to manage tasks efficiently. 
The application incorporates OTP-based login for enhanced security, and users can easily send their to-do lists to themselves via email with a single click.

Features:
OTP Login: Secure login with a One-Time Password sent to the user's email.
Task Management: Add, delete, update, and manage your to-do items effortlessly.
Email Functionality: Send your to-do list to your email with just a click of a button.
User Feedback: Users can provide feedback on the application to help improve its functionality.
Account Management: Manage user accounts, including updating personal information and changing passwords.

Tech stack:
Java: Backend logic and server-side processing.
JSP and Servlets: Dynamic web content generation and handling user requests.
SQL: Database management for storing user information, to-do items, and feedback.
Materialize CSS: Responsive and modern UI design.
